# Icon button

When the AI is acting through an icon button it wears the **non-text grain** — a dashed
"terminal" edge — the same inherited grade state `b-button` uses. It settles back to a
solid ink hairline on commit.

## In-transit

```html
<b-icon-button sym="/assets/sprite.svg#send" label="Sending…" data-commit="pending"></b-icon-button>
```
