# Keyboard hint

A quiet hairline chip that names a keyboard shortcut. One class — `.kbd`. Presentational
only: the real handler lives elsewhere (the ⌘K palette, a shell toggle). `keys` sets the
label.

## Examples

```html
<b-kbd keys="⌘K"></b-kbd>
<b-kbd keys="⌘B"></b-kbd>
<b-kbd keys="Esc"></b-kbd>
```
