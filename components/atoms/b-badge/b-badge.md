# Badge

One class — `.badge`. Tone is an attribute (`data-status`), not a second class.
Monochrome: state reads as ink vs faint, not hue (DESIGN-SYSTEM §2).

## Tones

### Active
```html
<span class="badge" data-status="active">Active</span>
```

### Archived
```html
<span class="badge" data-status="archived">Archived</span>
```
